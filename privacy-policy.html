Privacy Policy for ExplainIt!

Last Updated: February 18, 2026

Introduction

ExplainIt! ("we", "our", or "the extension") explains text selected by the user in Chrome. We are committed to minimizing data collection and being transparent about what is processed.

How ExplainIt Works

ExplainIt! works in direct provider mode. Selected text is sent from the extension to the AI provider chosen by the user.

Information Processed

Selected Text

  • You explicitly select text on a webpage.
  • The selected text is sent to your chosen AI provider to generate an explanation.
  • We do not store selected text on our servers.

Settings and API Keys

  • Local storage only: selected provider and provider API keys (chrome.storage.local).
  • Sync storage: language and tone preferences (chrome.storage.sync).
  • We do not have direct access to your browser local/sync storage.

Information We Do NOT Collect

  • Browsing history
  • Account identity data (name, email, phone)
  • Location data
  • Cookies or cross-site tracking identifiers
  • Analytics or telemetry from extension usage

Third-Party AI Providers

You choose which provider receives the selected text. Their terms and privacy policies apply to that processing. Supported providers include OpenAI, Anthropic, Google Gemini, and Groq.

  • OpenAI: https://openai.com/privacy
  • Anthropic: https://www.anthropic.com/privacy
  • Google: https://policies.google.com/privacy
  • Groq: https://groq.com/privacy-policy

Security

  • Provider API calls use HTTPS.
  • API keys are kept in local extension storage, not synced across devices by default.
  • UI output is rendered with safe text handling to reduce XSS risk.

Data Retention

  • Selected text: processed for request/response flow, not stored by us.
  • Explanation output: displayed in your browser.
  • Settings: stored in Chrome extension storage until you change/remove them.

Your Control

  • Change provider, language, and tone in extension settings.
  • Remove API keys from extension settings at any time.
  • Uninstall the extension to remove local extension data.

Permissions Explained

Permission Purpose
storage Store provider, API keys, and user preferences.
host_permissions Allow requests only to supported AI provider APIs.

Policy Updates

We may update this policy when product behavior changes. The date at the top indicates the latest revision.

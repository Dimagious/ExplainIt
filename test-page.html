How to Test

  1. Open Chrome DevTools (F12) and go to Console tab
  2. Select text in each test section
  3. Check console for ExplainIt! logs:
    • "Text selected" - for valid selections (≥3 chars)
    • "Selection too short" - for <3 chars
    • "Selection truncated" - for >2000 chars
  4. Verify iframe detection works

✅ Acceptance Criteria Checklist

  • ☐ Detects selection of ≥3 characters
  • ☐ Ignores selection of <3 characters
  • ☐ Truncates selection >2000 characters
  • ☐ Clears selection on deselect (click elsewhere)
  • ☐ Works in iframes
  • ☐ Logs appear in console

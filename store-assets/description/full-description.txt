🔍 ExplainIt! — Plain-English explanations of anything on the web

Highlight any sentence on a webpage, click once, and get a clear explanation right where you're reading — in English, Russian, Spanish, Chinese, Arabic, or 8 more languages.

Built for non-native English readers, students, and anyone consuming dense content (legal contracts, academic papers, news analysis, Stack Overflow, Reddit threads) without enough background to follow it word-for-word.

Free to use with Groq — no credit card, no signup beyond a single API key.

🚀 HOW IT WORKS

1) Select text on any webpage (3–2000 characters)
2) Click the floating ExplainIt bubble that appears next to your selection
3) Read the explanation in a popup right there in context
4) Copy with one click if needed — or switch tone / language and try again

🎯 THREE EXPLANATION DEPTHS

• Simple Words — clear plain English (or your chosen language) for any adult
• Like I'm 5 — short sentences, no jargon, friendly comparisons
• Expert — preserves technical terms and key nuance

🔒 PRIVACY

• Your API key is stored locally on this device only — never synced to Google, never sent to any ExplainIt server
• Only text you explicitly select is processed
• Selected text is sent only to the AI provider you chose
• No browsing history collection
• No analytics, no telemetry, no signup, no email, no account
• ExplainIt does not run its own backend — there is no ExplainIt server in the loop between you and your chosen provider
